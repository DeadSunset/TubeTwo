# Offline myTube (Live Server only)

Статическое браузерное приложение без Node.js и серверов.

## Запуск

1. Откройте проект в VS Code.
2. Запустите `Live Server` для `index.html`.
3. Используйте кнопки:
   - Добавить общую папку
   - Добавить канал
   - Добавить видео

## Что реализовано

- Хранение данных в IndexedDB (`sources`, `channels`, `videos`, `thumbs`, `comments_user`, `comments_imported`, `errors_logs`).
- Источники через File System Access API.
- Рекурсивное сканирование папок.
- Определение shorts по длительности и пропорциям.
- Главная лента, каналы, страница канала, страница видео, shorts, профиль.
- Сохранение прогресса видео каждые 5 секунд.
- Лайки, просмотренные, понравившиеся.
- Пакетные операции (по 10) с паузой/продолжением/отменой.
- Генерация превью WebP 320px.
- Логи ошибок, экспорт/импорт бэкапа (JSON-версия).

## Примечание

ZIP-бэкап и полноценный HTML-матчинг канала в этой версии упрощены до JSON/базового импорта.
