# Offline myTube (Live Server only)

Полностью статический фронтенд без Node.js и серверной части.

## Запуск

1. Откройте проект в VS Code.
2. Запустите Live Server на `index.html`.
3. Разрешите доступ к папкам через File System Access API.

## Что исправлено в этой версии

- Переработан UI (типографика, размеры, сетки, карточки, панели) ближе к предоставленным референсам.
- Исправлен рекурсивный импорт из источников:
  - `common`: файлы в корне -> `Общий канал`, подпапки 1-го уровня -> отдельные каналы.
  - `channel`: вся папка -> один канал с деревом плейлистов.
- Добавлены реальные прогресс-панели задач:
  - сканирование,
  - генерация превью (batch по 10),
  - массовый импорт по каналу.
- Добавлены `pause / resume / cancel` для очередей.
- Добавлена проверка доступности источников и кнопка «Восстановить доступ» в профиле.

## Ограничения

- Бэкап пока в JSON (не ZIP).
- HTML/URL импорт пока в упрощённом режиме.
